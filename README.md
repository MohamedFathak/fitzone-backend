# FitZone Gym Management System - Backend

## Description
This is the Spring Boot backend for the FitZone Gym Management System. It provides REST APIs for managing users, memberships, gym classes, bookings, and authentication.

## Technologies Used
- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- MySQL
- Maven
- REST API

## Features
- User Registration & Login
- JWT Authentication
- Membership Management
- Gym Class Management
- Booking Management
- Secure REST APIs
